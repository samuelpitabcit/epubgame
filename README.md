# E-Type
A fun terminal-based typing game that helps users improve speed and accuracy by typing content from text or EPUB files, It also helps with studying, since players read the material as they type.

https://github.com/samuelpitabcit/epubgame

## Prerequisites
*Java 17** or Higher
*Maven** for dependency management and building the project

## Dependencies
The project uses the following libraries:

- [Apache Tika](https://tika.apache.org/) for EPUB parsing
- [JNativeHook](https://github.com/kwhat/jnativehook) for global keyboard input handling

## Installation
first clone the project:
``` bash
git clone https://github.com/samuelpitabcit/epubgame
```
cd project into the project
``` bash
cd placeholder\epubgame
```
Build the project using Maven:
``` bash
mvn clean package
```
Run the Applicaton
``` bash
java -jar target/HttpHacks2025-1.0-SNAPSHOT.jar
```
## Description
On the Terminal, Select what book you want to read, to stop typing, press ESCAPE
